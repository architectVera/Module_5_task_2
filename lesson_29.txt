Домашне завдання - лекція 29
============================

Ініціалізувати Django-проект (з цим проектом будемо працювати до кінця курсу).
Створити два Django-застосунки (`startapp`): task, user.

Завдання
--------

- за URL `task/` повина відображатись сторінка з усіма наявними тасками
- за URL `task/<id>/` повина відображатись сторінка окремого таска
- за URL `register/` повина відображатись сторінка реєстрації нового
  користувача
- за URL `login/` повина відображатись сторінка логіна користувача

Вимоги
------

- Кожна сторінка має містити навігацію за сайтом: посилання на список тасків,
  сторінку реєстрації та логіна.
- У списку, ті таски, що відмічені як "виконані" повині мати відмітку про те

Приклади
--------

Приблизна структура таска:

```
{
    "id": ..., (int або uuid)
    "title": "Descriptive task title",
    "description": "Some long task description...",
    "created": "2022-01-24 12:00",
    "updated": "2022-01-25 12:00",
    "completed": False
}
```
